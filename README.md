# 💼 InvoiceApp - Free Invoice & Accounting

Free invoicing application for Cameroon SMEs.

## ✨ Features
- 📄 Create & manage invoices
- 💸 Track expenses
- 📊 Dashboard analytics
- 📱 Mobile friendly
- 💯 Completely free

## 🚀 Quick Start

### Prerequisites
- Node.js v14+ (Download from nodejs.org)

### Installation

1. **Extract the project folder**
2. **Open terminal in the folder**
3. **Install dependencies:**
```bash
npm install
```

4. **Start the app:**
```bash
npm start
```

5. **Open in browser:**
```
http://localhost:3000
```

## 💰 Monetization

### Free Tier
- Up to 5 invoices/month
- Basic expense tracking

### Pro Tier ($3/month)
- Unlimited invoices
- Advanced features

## 📚 Usage

1. **Login:** Use any email and password
2. **Create Invoice:** Fill in client info and items
3. **Track Expenses:** Log business expenses
4. **View Dashboard:** See profit/revenue overview
